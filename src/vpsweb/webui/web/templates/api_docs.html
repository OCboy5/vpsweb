Poems API

GET /api/v1/poems/

List all poems with optional filtering and pagination

POST /api/v1/poems/

Create a new poem in the repository

GET /api/v1/poems/{poem_id}

Get detailed information about a specific poem

PUT /api/v1/poems/{poem_id}

Update an existing poem

DELETE /api/v1/poems/{poem_id}

Delete a poem and all its translations

GET /api/v1/poems/{poem_id}/translations

Get all translations for a specific poem

Translations API

GET /api/v1/translations/

List translations with filtering and pagination

POST /api/v1/translations/

Create a new translation (human or AI)

GET /api/v1/translations/{translation_id}

Get detailed information about a translation

PUT /api/v1/translations/{translation_id}

Update an existing translation

DELETE /api/v1/translations/{translation_id}

Delete a translation and its associated data

POST /api/v1/translations/trigger

Trigger AI translation workflow

Statistics API

GET /api/v1/statistics/overview

Get comprehensive repository statistics

GET /api/v1/statistics/poems/language-distribution

Get language distribution statistics

GET /api/v1/statistics/search/metrics

Get search-related metrics and filters

GET /api/v1/statistics/translators/productivity

Get translator productivity metrics

GET /api/v1/statistics/timeline/activity

Get repository activity timeline

Response Formats

Success Responses

HTTP 200 OK
Content-Type: application/json

{
  "id": "01H...",
  "poet_name": "William Shakespeare",
  "poem_title": "Sonnet 18",
  "created_at": "2025-01-15T10:30:00Z",
  ...
}

Error Responses

HTTP 404 Not Found
Content-Type: application/json

{
  "detail": "Poem with ID '01H...' not found"
}

Validation Errors

HTTP 422 Unprocessable Entity
Content-Type: application/json

{
  "detail": [
    {
      "loc": ["body", "poet_name"],
      "msg": "field required",
      "type": "value_error.missing"
    }
  ]
}

Usage Examples

Create a New Poem

curl -X POST "http://localhost:8000/api/v1/poems/" \
  -H "Content-Type: application/json" \
  -d '{
    "poet_name": "William Shakespeare",
    "poem_title": "Sonnet 18",
    "source_language": "English",
    "original_text": "Shall I compare thee to a summer'\''s day?",
    "metadata": "{\"year\": 1609}"
  }'

Get All Poems

curl -X GET "http://localhost:8000/api/v1/poems/" \
  -H "Accept: application/json"

Create a Translation

curl -X POST "http://localhost:8000/api/v1/translations/" \
  -H "Content-Type: application/json" \
  -d '{
    "poem_id": "01H...",
    "target_language": "Chinese",
    "translator_name": "AI Translation System",
    "translated_text": "我能否将你比作夏日？",
    "quality_rating": 4
  }'

Get Statistics

curl -X GET "http://localhost:8000/api/v1/statistics/overview" \
  -H "Accept: application/json"
